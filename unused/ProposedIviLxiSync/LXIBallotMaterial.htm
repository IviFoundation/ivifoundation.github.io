Proposed Resolution

The technical committee adopts the IVI-3.15 LxiSync specifcation as expressed in:

  • the IVI Specification 3.15 as expressed in IVI-3.15_LxiSync_Prelim_2006-01-25.doc, posted at here. 
  • The changes to IVI-3.1 and IVI-1.2 that provide for the addition of this capability and will simplify adding future classes to IVI described in the document Proposed Changed to IVI 3.1 and IVI 1.2 for IVI LxiSync 2006-01-13 posted here. 
  • The IviLxiSync shared components posted on the IVI web site here

In addition, the technical committee resolves to maintain a list of IVI GUIDs and Help Context IDs on the IVI web site as it currently exists here.
